## 冻结结论

- 前端遵循 `Host -> Shell -> Feature -> Foundation`。
- Rust 遵循 `protocol -> runtime -> service/control-plane`。
- `src-tauri` 只保留桌面宿主桥、命令注册和打包配置。
- `docker`、`k8s` 作为 `server` 交付形态目录，不创建第二套业务模型。
