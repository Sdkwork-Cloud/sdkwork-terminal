# 2026-04-10 v0.2.40 - Session Attach/Detach/Reattach IPC

## Changed

- Added `desktop_session_attach / detach / reattach` to the desktop bridge, so Step 06 Session write semantics are no longer trapped inside Rust-only `SessionRuntime`.
- `@sdkwork/terminal-infrastructure` now exposes `attachSession(...)`、`detachSessionAttachment(...)`、`reattachSession(...)` for desktop consumers.
- `@sdkwork/terminal-shell` now best-effort detaches still-live attachments during host teardown.
- `sdkwork-terminal-session-runtime` now keeps a no-attachment session in `Detached` even if background output continues to be recorded.
