# 2026-04-10 v0.2.41 Desktop Connector Reattach Alignment

## Summary

This round closes a desktop product mismatch in Step 07: the bridge already supported generic `desktop_session_reattach`, but Session Center still exposed interactive reattach only for `local-shell`. Desktop now reattaches `ssh / docker-exec / kubernetes-exec` sessions through the same real session-first path, while `remote-runtime` remains read-only until its live attach path lands.

- `apps/desktop/src/DesktopSessionCenterOverlay.tsx`
