# 2026-04-10 v0.2.40 Step 07 Desktop Connector Catalog Product Entry

- `src-tauri` 新增 `desktop_execution_target_catalog`，桌面宿主现可显式输出 execution target catalog 读模型。
- `@sdkwork/terminal-infrastructure` 新增 `executionTargets()`，`apps/desktop/src/resource-center.ts` 现可将 desktop bridge catalog 投影为 `ResourceCenterSnapshot`。
- `apps/desktop/src/App.tsx` 现持有 `resourceCenterSnapshot` 状态，并据此派生顶栏 `Connectors` 菜单项与 connector launch intent。
- `apps/desktop/src/connector-shell.ts` 改为显式消费当前 snapshot，不再依赖模块加载期静态 `createResourceCenterSnapshot()` 回退。
