# 2026-04-11 v0.2.40 Shell Snapshot State Cache

## Summary

The shell model now caches the whole `TerminalShellSnapshot` by state reference, not just per-tab snapshots.

## Changes

- Added a shell-level `WeakMap` cache keyed by `TerminalShellState`.
- Reused the cached shell snapshot when the same state object is read repeatedly.
- Extended tests to require shell snapshot object reuse and `tabs` array reuse for unchanged state reads.

## Impact

- Cuts repeated snapshot envelope allocation on hot render paths.
- Reduces repeated `tabs` array creation and `activeTab` lookup for identical state reads.
- Complements the existing per-tab snapshot cache without changing terminal behavior.
