# 2026-04-10 v0.2.40 Session Center Overlay Reattach

## Summary

- 桌面端新增 terminal-first `Session Center` overlay，入口位于 shell dropdown，不再需要 dashboard/home 页面。
- Session Center 现在可以对 `local-shell` detached session 执行真实 `desktop_session_reattach`，成功后由 ShellApp 打开新的 runtime-backed tab 并继续 replay/live 链路。
- 非 `local-shell` session 当前仍只展示会话真相、replay 与诊断，不暴露伪交互式重连按钮。
