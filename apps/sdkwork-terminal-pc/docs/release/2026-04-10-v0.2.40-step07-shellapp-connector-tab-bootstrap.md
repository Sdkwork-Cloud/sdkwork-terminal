# 2026-04-10 v0.2.40 Step 07 ShellApp Connector Tab Bootstrap

## Changed

- `packages/sdkwork-terminal-shell/src/model.ts` 新增 tab 级 `runtimeBootstrap`，desktop tab 不再被硬编码为 `local-shell`。
- `packages/sdkwork-terminal-shell/src/index.tsx` 新增 desktop connector bootstrap 分支，ShellApp 现可在 `createLocalShellSession(...)` 与 `createConnectorInteractiveSession(...)` 之间按 tab 事实切换。
- connector interactive session 现可被承接为顶部 live terminal tab。
- desktop 输入热路径新增 `idle / binding / retrying` 排队语义，bootstrap 早期输入不再直接丢失。
