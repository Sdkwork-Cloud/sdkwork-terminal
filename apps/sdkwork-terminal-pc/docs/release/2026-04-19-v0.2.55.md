# 2026-04-19 v0.2.55

## Release Strategy

- This release advances the shipped `v0.2.54` line by removing the now-obsolete Node24 compatibility override from CI and release workflows.
- The goal is to prove that the repository's GitHub Actions baseline is clean on its own, without depending on transitional runtime-forcing configuration.

## Wave / Step

- Wave B
- Release engineering hardening
- Remove obsolete GitHub Actions runtime override

## Changed

- Removed `FORCE_JAVASCRIPT_ACTIONS_TO_NODE24` from `.github/workflows/ci.yml`.
- Removed `FORCE_JAVASCRIPT_ACTIONS_TO_NODE24` from `.github/workflows/release-reusable.yml`.
- Updated the workflow contract test to assert that both workflows remain free of the removed override.

## Fixed

- Fixed the remaining temporary compatibility scaffolding in the GitHub Actions baseline after the action-version upgrades delivered in `v0.2.54`.
- Fixed a workflow-standard ambiguity where repository automation still appeared to depend on an override that should no longer be part of the supported release design.
