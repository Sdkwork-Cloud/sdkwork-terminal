# 2026-04-10 v0.2.40 Step 07 Session-First Runtime Write Surface

## Summary

收口 Step 07 桌面 terminal 写入/订阅链路的命名漂移。桌面桥接、ShellApp 与 Tauri IPC 现统一使用 `session-first` 语义，不再把 live terminal 写链路绑定到 `local-shell` 命名。

## Changed

- `packages/sdkwork-terminal-infrastructure` 新增并冻结 `writeSessionInput / writeSessionInputBytes / resizeSession / terminateSession / subscribeSessionEvents`。
- 旧 `writeLocalShellInput / writeLocalShellInputBytes / resizeLocalShellSession / terminateLocalShellSession / subscribeLocalShellSessionEvents` 保留为兼容别名，但不再是产品主命名。
- `packages/sdkwork-terminal-shell` 全量切到 `session-first` API；connector tab 与 local-shell tab 共享同一套写入、resize、terminate、订阅语义。
- `src-tauri` 新增 `desktop_session_input / desktop_session_input_bytes / desktop_session_resize / desktop_session_terminate` 及配套 capability/permission。
- README、架构、step 与循环 prompt 已同步改写为新的命名事实。
