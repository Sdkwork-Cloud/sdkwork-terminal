# 2026-04-10 v0.2.44 Step 07 Connector Batch Smoke Matrix

## Summary

This round promotes the Step 07 connector smoke tool from single-template output to a repository-owned batch execution matrix. The project can now emit a concrete `platform-target` evidence plan for `windows-desktop / ubuntu-desktop / macos-desktop x ssh / docker-exec / kubernetes-exec`, making it explicit which smoke jobs can run in parallel and which Step/release closeout actions must remain serial.

- `tests/connector-interactive-probe.test.ts`
