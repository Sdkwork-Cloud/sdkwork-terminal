# release 文档目录

- 本目录存放版本说明、`CHANGELOG.md`、阶段发布记录与商业化交付记录。
- 历史版本记录按当时事实保留；若历史措辞与当前架构标准存在差异，以 `README.md`、`docs/架构/`、`docs/step/` 的最新标准为准。
- 命名规则：`YYYY-MM-DD-vX.Y.Z.md`
- `0.x.y` 用于 `Step 00-12` 阶段；首个完整商业化闭环版本为 `1.0.0`。
- 每次真实能力增量都必须记录：
  - 日期
  - 版本
  - 对应波次 / step
  - Changed / Fixed / Verified
  - 架构回写
  - 风险与回滚
