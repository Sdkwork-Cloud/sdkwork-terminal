# Step 04 主链闭环审计

## 当前事实

- `src-tauri` 已收口为薄宿主，只保留 desktop bridge 命令注册、状态装配与启动入口。
- `crates/sdkwork-terminal-session-runtime` 已提供 `LocalSessionDaemon` 与健康快照模型。
- `apps/desktop + @sdkwork/terminal-infrastructure` 已打通 readiness 拉取、daemon 健康查看和 `start / reconnect / stop` 控制。

## 等级判定

- 当前能力等级：`L3`
- 判定理由：桌面跨进程主路径已可用，但 PTY 热路径、attach/replay、恢复与回滚口径尚未达到 `L4`

## 剩余差距

- 尚未接入 `terminal-core` 与本地 runtime 热路径。
- 尚未具备 Session attach / detach / replay 真实语义。
- 当前 observability 仍以摘要快照为主，指标上报与诊断包要在后续步骤完善。
