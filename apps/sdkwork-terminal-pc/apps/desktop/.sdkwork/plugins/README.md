# Plugins

This directory stores checked-in plugins for the `sdkwork-terminal-pc` desktop sub-surface.

Plugins must follow [`../../../../../../sdkwork-specs/SDKWORK_WORKSPACE_SPEC.md`](../../../../../../sdkwork-specs/SDKWORK_WORKSPACE_SPEC.md). Installable plugins must provide `.codex-plugin/plugin.json` and document contributed skills, tools, apps, scripts, or MCP servers.
